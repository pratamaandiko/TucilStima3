Tucil Stima 3


Untuk menambahkan mark pada peta:
1. Klik pada peta
2. Jika sudah selesai, Klik tombol "Selesai"
3. Klik tombol "Hapus mark" untuk menghapus semua mark
Untuk menghubungkan mark pada peta:
1. Klik beberapa mark yang ingin dihubungkan
2. Tekan tombol "Hubungkan"
3. Jika sudah selesai, Klik tombol "Selesai"
4. Klik tombol "Hapus mark" untuk menghapus semua mark
